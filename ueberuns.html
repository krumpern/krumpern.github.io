Wie alles begann...

...in einer Reihe im Comuputerraum an der Universität Innsbruck formierte sich ein Trio um die Probleme der Welt mittels HTML, JAVA und CSS zu bekämpfen. Ihre Motivation war die Lehrveranstaltung: Geoinformatik: Web Mapping des Masterstudiums Geographie und dafür eine interaktive, smarte und stylische Website zu erstellen.

Informationen zur Lehrveranstaltung findet man hier.

Die Daten der Themenwege wurden als Open Source Dateien von data.gv.at entnommen.
Alle Bilder wurden von pixabay.com bezogen.
